    Opis

    Dokumentacja programu z lab 3 - Przetwarzanie obrazów 1

    Kod

    Program został napisany zgodnie ze standardem c99, kompilator GNU GCC.

    Stworzne zostały dwie wersje programu, polecam skorzystać z wersji DEBUG dla szybszego sprawdzenia wszystkich funkcji.

    Kompilacja 

    gcc -std=c99 -Wall -pedantic File.c -lm -o out/Main.out Main.c

    Kompilacja DEBUG

    gcc -std=c99 -Wall -pedantic File.c -lm -o out/MainDEBUG.out Main.c DEBUG=1

    Uruchamianie 

    ./Main.out

    Uruchamianie DEBUG

    ./MainDEBUG.out kubus.pgm

    Testy

    Testy wszystkie zostały diametralnie skrócone dzięki trybie DEBUG, wszystkie operacje na plikach zostają wykonane jednocześnie zatem rezultaty wrzucam jako jeden test.

    Test 1

    Operacje wykonane są opisane, dodane zostały także komentarze w plikach.

    Dane

    ./MainDEBUG.out kubus.pgm

    Test 2

    Wszystkie funkcje pobierające z klawiatury bądź wymagające integracji użytkownika zostały zaprojektowane w taki sposób, że jeżeli oczekiwana wartość jest nieprawidłowa program pyta do skutku.

    Przykład takiego zastosowania

    do{
  printf("Enter scope [0-100]: ");
  scanf("%d", &scope);
} while (scope > 100 || scope < 0);

    Test 3

    Podanie nazwy pliku nie wymaga dodania formatu, lecz jest to sprawdzane i poprawiane. Tzn. Jeżeli podamy nazwe "kubus" program automatycznie doda rozszerzenie .pgm, możemy także podać nazwę "kubus.pgm", obie formy są poprawne.

    Poprawianie nazwy pliku

    char *tester = strchr(fileName, *".pgm");
    if (tester == NULL) strcat(fileName, ".pgm");
